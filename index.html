An HTML5 Swarmatron-like Sound Synthesis Unit

The Swarmatron is an electronic music instrument designed by Brian Dewan and Leon Dewan of Dewanatron. There are currently 85 units in existence. Each Swarmatron has been built by hand.

The following digital implementation is a rough emulation of a Swarmatron coded for the purpose of exploring some of the capabilities the Web Audio API offers. It is coded with pure vanilla javscript (using only Modermizr for Web Audio capability detection).

If you don't see a Swarmatron directly above, it is because your browser does not support Web Audio. Many modern browsers, among them current versions of Chrome, Firefox and Safari, do support Web Audio.
